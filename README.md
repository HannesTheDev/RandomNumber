# RandomNumber
Simply get a random number by command, you can easily set in the settings in which range the random number should be selected.

## Commands
Commands | Description | Default | Aliases | Permissions
--------- | ------------------- | ------- | ---------- | -----------
``/randomnumber`` | ``Generate a random number`` | ``op`` | ``randomn, rnumber, rn`` | ``randomnumber.randomnumber.cmd``
